# Ze-Agenda ZCPS

Mini-hub de réservation semi-automatisé pour ZCPS.

## Fonctionnalités
- Génération automatique des jours à venir.
- Affichage des créneaux en français.
- Blocage des créneaux par date.
- Intégration avec Google Sheets (via `SHEET_URL` dans `app.js`).
- Prêt pour GitHub Pages.

## Installation
1. Clonez le dépôt.
2. Configurez votre `SHEET_URL` dans `app.js`.
3. Hébergez sur GitHub Pages.
